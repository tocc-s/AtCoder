## 使用言語
- C++
- JavaScript (Node.js)
- TypeScript

後々別の言語でも解いてみるかも。

## 使い方
- コンパイル (g++)
```
  g++ -std=c++17 [name].cpp -o [name]
```
- 実行 (g++)
```
  ./[name]
```
- 実行 (JavaScript node)
```
  node [name].js
```
- 実行 (TypeScript)
```
  ts-node [name].ts
```
- コンパイル (TypeScript -> JavaScript)
```
  tsc [name].ts
```
- 基本的にAtCoderやAOJなどの参加コンテスト、過去問を解いてから追加。
- 参加中のコンテストはコンテスト終了後に追加。
- 他にもアルゴリズム学習に関する問題もメモとして追加する予定。
