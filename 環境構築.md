# 環境構築

## 開発環境構築手順
  - JavaScript
  - TypeScript

# WSL
- JavaScript
  1. 最新安定板Node.jsを導入
  ```
    sudo apt install -y nodejs npm
  ```
  2. インストールしたnpmを利用してn packageを導入
  ```
    sudo npm install n -g
    sudo n stable
  ```
  3. バージョン確認
  ```
    node -v
    > v14.17.0

    npm -v
    > 3.5.2
  ```
  <br>
- TypeScript
  1. JavaScriptの手順でnpm導入まで行う
  2. TypeScript、ts-nodeをインストール
  ```
    npm install -g typescript ts-node
  ```
  3. バージョン確認
  ```
    tsc -v
    > Version 4.3.2
    
    ts-node -v
    > v10.0.0
  ```
